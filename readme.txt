This code implements a Bug1 algorithm in Python using Shapely for geometry handling and Matplotlib for visualization.It allows a robot to navigate around polygonal obstacles within a bounded workspace, generating animated .gif visualizations of its path.

Install all required Python packages via pip:
pip install shapely matplotlib pillow numpy
Python 3.8+ recommended

It can also run on Google Colab

Select the obstacle file number
Enter the robot radius
Enter start and goal coordinates (or enable random generation inside the code Line # 470 USE_RANDOM = True)
